git init 
(sirve para iniciar el repositorio local)

git config --local user.name "jeancuy"
(esto configura el nombre usuario localmente)

git config --local user.email "jeancarlosmulato513@gmail.com"
(esto configura al email de usuario asociado a la cuenta GitHub)

git branch -M main
(cambiar el nombre de la rama de Master a Main)

git add . 
(añade todos los archivos al repositorio)

git commit -m "version de subida"
(es la versión que usted esta subiendo al repo)

git push -u origin main
(este empuja los cambios locales a la nube)

git status
(combrueba el estado de la rama)

git remote add origin https://github.com/jeancarlosmulato513-alt/maquetado_web.git
(añade la dirección de su repositorio se hace o ejecuta una unica vez)
